# TaskFlow – Task Manager System - Angular-ASP.NET-Core-SQL-Server

A task management system that helps users organize tasks by categories, track progress, and view productivity insights through a dashboard.

---

## Project Overview
This system solves the problem of unstructured task tracking by providing categorized task management and visual progress insights.
It is designed for individuals who want clarity over their daily tasks and productivity.
The project exists as a practical example of a real-world, scalable task management system.

---

## User Roles & Flows
### User
- Create, update, and delete tasks
- Organize tasks into categories
- Track task completion status
- View dashboard statistics and trends

---

## Features
- Task creation and management
- Category-based task organization
- Dashboard with charts and statistics
- Visual productivity insights (bar, pie, sparkline charts)
- Clean and intuitive UI flow

---

## Tech Stack
**Frontend**
- Angular
- TypeScript
- SCSS
- Angular Router

**Backend**
- ASP.NET Core Web API
- C#
- Entity Framework Core

**Database**
- SQL Server

**Architecture & Security**
- Service and Repository pattern
- SOLID principles
- DTO-based API contracts

---

## Architecture & Design
The system follows a clean layered architecture using Controllers, Services, and Repositories.
DTOs are used to separate API contracts from domain models.
The design emphasizes SOLID principles to ensure scalability, maintainability, and clear separation of concerns.
